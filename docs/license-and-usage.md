# License and Usage

This dataset is intended for research, teaching, and non-commercial experiments.

Allowed uses include:

- inspecting and evaluating infrared target elements
- building non-commercial research prototypes
- using the data in teaching materials
- citing the dataset in academic or technical work

Restricted uses include:

- selling the dataset or modified copies
- redistributing the full archive under another name
- using the dataset in commercial products or services without permission
- presenting the dataset as a complete benchmark for all infrared imaging cases

The generation program and its source code are not included in this repository.
